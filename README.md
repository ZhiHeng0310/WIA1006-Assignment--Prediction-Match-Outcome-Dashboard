# 💖 Match Success Predictor

A machine learning-powered web application that predicts the probability of success in a dating match using a Logistic Regression model.

## 📌 Features
- 📊 Predict match success probability
- 🧠 Logistic Regression model (custom implemented)
- 🎯 Confidence level and key contributing factors
- 💻 Interactive and responsive UI (React + Tailwind CSS)

---

## 🛠️ Tech Stack
- HTML, CSS (Tailwind)
- JavaScript (React via CDN)
- Logistic Regression (custom implementation)

---

## 📈 Model Details
- Algorithm: Logistic Regression
- Accuracy: ~54%
- Balanced class handling
- 26 input features (demographics, behavior, profile quality)

---

## 🧪 How It Works
1. User inputs personal and app behavior data
2. Features are scaled and processed
3. Logistic Regression computes probability
4. Output:
   - Success / Not Success
   - Probability %
   - Confidence level
   - Key influencing factors
